# Harmonic Analyzer

A powerful web-based tool for analyzing musical chords and harmonies.

## Live Demo

Try it out here: [Harmonic Analyzer](https://linkcrimson.github.io/harmonic_analyzer/)

## Features

- Real-time chord analysis
- MIDI keyboard support
- Visual piano interface
- Detailed harmonic information

## Development

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
